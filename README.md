# drinkery
App to find the recipe of your favorite drink.

[Wireframes for this project](Drink_Up.pdf)

User Stories

1. The user should be able to enter an existing username and password, click the Log In button and be routed to the main page.

2. The user should be able to click on the Register tab, enter a valid username and password, click the Submit button and be routed to the main page.

3. The user should be able to click a category button (Favorites, Drinks, Liquors, Mixers) and be routed to the corresponding page.

4. The user should be able to click on a drink in the Favorites list and be routed to the corresponding recipe page for that drink.

5. The user should be able to click on a drink in the Drinks list and be routed to the corresponding recipe page for that drink.

6. The user should be able to click on a liquor in the Liquors list and be routed to the Drinks page, where the drinks containing that chosen liquor will be listed. The user should then be able to click on a drink and be routed to the corresponding recipe page for that drink.

7. The user should be able to click on the mixer in the Mixers list and be routed to the Drinks page, where the drinks containing that chosen mixer will be listed. The user should then be able to click on a drink and be routed to the corresponding recipe page for that drink.

8. The user should be able to click on the Save to Favorites button and have the listed drink saved to their favorites list. The drink should then appear on the Favorites page.

9. The user should be able to click on the Remove from Favorites button and have the listed drink removed from their favorites list. The drink should not appear on the Favorites page.
